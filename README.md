# Reto técnico

## Ordenar los nombres de las bandas sin los artículos

```
https://retos-tecnicos-backend.lizzie136.now.sh/bands
``` 

Luego muéstralos en el HTML en orden alfabético, pero ignorando los artículos como 'The', 'A', 'An'.
